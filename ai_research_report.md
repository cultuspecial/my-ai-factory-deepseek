# AI 简报 (2026-06-05)

### HANDOFF: Humanoid Agentic Task-Space Whole-Body Control via Distilled Complementary Teachers
📄 HANDOFF提出一种紧凑、直观、通用且具表达力的任务空间接口，并通过多教师知识蒸馏与上下文条件门控机制，将全身运动跟踪、行走和跌倒恢复三个互补专家技能融合为单一人形机器人全身控制器。
🔗 http://arxiv.org/abs/2606.06493v1

### Code2LoRA: Hypernetwork-Generated Adapters for Code Language Models under Software Evolution
📄 论文提出Code2LoRA框架，通过超网络生成代码仓库专用的LoRA适配器，无需在推理时增加额外token即可注入仓库上下文知识，并支持静态快照与动态演化的代码库适配场景。
🔗 http://arxiv.org/abs/2606.06492v1

### TempoVLA: Learning Speed-Controllable Vision-Language-Action Policies
📄 **TempoVLA 提出了首个通过显式条件控制单一视觉-语言-动作模型（VLA）执行速度的方法，解决了现有模型只能以固定速度运行、无法在高速低风险与低速高精度阶段切换的局限。**
🔗 http://arxiv.org/abs/2606.06491v1

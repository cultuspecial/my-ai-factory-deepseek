# AI 简报 (2026-04-28)

### Personalized Worked Example Generation from Student Code Submissions using Pattern-based Knowledge Components
📄 该论文提出了一种基于程序抽象语法树提取学生代码中结构化知识组件模式的方法，能够自动为每位学生生成个性化示例，从而减少人工编写教学案例的工作量，并更精准地针对学生的逻辑错误与部分解决方案提供定制化学习内容。
🔗 http://arxiv.org/abs/2604.24758v1

### Learning to Think from Multiple Thinkers
📄 这篇论文的贡献在于：在密码学假设下，揭示了在被动数据收集场景中，即使使用来自多个不同思维者的正确思维链监督，学习仍可能很困难；但提出了一个通用的高效主动学习算法，能够在多项式和少数思维者条件下实现可学习性。
🔗 http://arxiv.org/abs/2604.24737v1

### Learning to Rotate: Temporal and Semantic Rotary Encoding for Sequential Modeling
📄 该论文提出将旋转位置编码（RoPE）中的旋转流形从固定先验结构拓展为可学习的、由输入信号条件化的语义旋转空间，从而为注意力机制开辟了一个与语义嵌入正交的新表达维度。
🔗 http://arxiv.org/abs/2604.24717v1

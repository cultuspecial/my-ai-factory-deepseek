# AI 简报 (2026-02-08)

### Shared LoRA Subspaces for almost Strict Continual Learning
📄 本文提出Share方法，通过构建并动态更新一个共享的低秩子空间，实现无需数据回放或使用多个适配器的严格持续学习，有效缓解灾难性遗忘并跨任务与模态高效整合知识。
🔗 http://arxiv.org/abs/2602.06043v1

### DyTopo: Dynamic Topology Routing for Multi-Agent Reasoning via Semantic Matching
📄 DyTopo提出了一种动态拓扑路由的多智能体推理框架，通过语义匹配在每轮推理中重构稀疏通信图，从而根据任务阶段需求自适应地路由消息，显著提升了多轮推理性能。
🔗 http://arxiv.org/abs/2602.06039v1

### CommCP: Efficient Multi-Agent Coordination via LLM-Based Communication with Conformal Prediction
📄 该论文提出CommCP框架，通过结合大语言模型与共形预测技术，在多智能体协作任务中实现高效、可靠的通信与决策，从而提升异构机器人团队在复杂场景下的信息获取与任务执行效率。
🔗 http://arxiv.org/abs/2602.06038v1

# AI 简报 (2026-01-30)

### RedSage: A Cybersecurity Generalist LLM
📄 RedSage：首个通过大规模网络安全领域持续预训练与专家工作流模拟微调数据构建的开源、可本地部署的网络安全通用大语言模型，解决了隐私风险与领域适应性的双重挑战。
🔗 http://arxiv.org/abs/2601.22159v1

### Hybrid Linear Attention Done Right: Efficient Distillation and Effective Architectures for Extremely Long Contexts
📄 本文提出HALO方法，通过高效的参数转换与知识蒸馏，将预训练的Softmax注意力模块转化为线性注意力模块，构建混合注意力架构，在显著降低长上下文推理成本的同时，有效保持了模型性能。
🔗 http://arxiv.org/abs/2601.22156v1

### Exploring Reasoning Reward Model for Agents
📄 本文提出了一种面向智能体的推理奖励模型（Agent-RRM），通过提供结构化反馈（包括显式推理轨迹、聚焦式批评和改进指导、以及整体评分）来优化智能体训练，弥补了传统稀疏结果奖励在区分中间推理质量上的不足。
🔗 http://arxiv.org/abs/2601.22154v1

# AI 简报 (2026-06-04)

### Streaming Communication in Multi-Agent Reasoning
📄 StreamMA通过将推理步骤流式传输至下游智能体，实现了多智能体推理的流水线并行，在显著降低延迟的同时，利用早期更可靠的推理步骤避免了后期错误误导，首次通过闭式联合分析证明了其效果优于串行与单次协议。
🔗 http://arxiv.org/abs/2606.05158v1

### Reinforcement Learning from Rich Feedback with Distributional DAgger
📄 该论文提出了一种基于分布化DAgger算法的强化学习方法，能够有效利用执行轨迹、工具输出、专家修正等丰富反馈信号，通过前向交叉熵目标实现序列级别的细粒度信用分配，从而克服传统RLVR仅依赖二元正确性奖励的局限性。
🔗 http://arxiv.org/abs/2606.05152v1

### Multi-Column RBF Neural Network Using Adaptive and Non-Adaptive Particle Swarm Optimization
📄 该论文提出了一种结合自适应与非自适应粒子群优化的多列径向基函数神经网络，在保持全局搜索能力的同时，显著提升了大规模数据集上的训练效率与精度。
🔗 http://arxiv.org/abs/2606.05150v1

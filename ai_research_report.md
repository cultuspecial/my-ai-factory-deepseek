# AI 简报 (2026-01-31)

### RedSage: A Cybersecurity Generalist LLM
📄 RedSage是一个开源的、可本地部署的网络安全大语言模型，其贡献在于通过大规模网络安全领域数据（11.8B token）的持续预训练，并结合模拟专家工作流程生成的26.6万条多轮对话数据进行监督微调，实现了在保护数据隐私前提下的专业化网络安全任务适应能力。
🔗 http://arxiv.org/abs/2601.22159v1

### Hybrid Linear Attention Done Right: Efficient Distillation and Effective Architectures for Extremely Long Contexts
📄 本文提出HALO方法，通过高效蒸馏与改进架构设计，在极长上下文建模中实现了高性能与高效率的平衡，显著降低了混合注意力模型的训练成本并提升了长上下文性能。
🔗 http://arxiv.org/abs/2601.22156v1

### Exploring Reasoning Reward Model for Agents
📄 本文提出了一种多方面的智能体推理奖励模型（Agent-RRM），通过提供结构化反馈（包括显式推理轨迹、针对性批判和改进指导以及整体评分）来优化智能体强化学习，克服了传统稀疏结果奖励无法区分中间推理质量的局限。
🔗 http://arxiv.org/abs/2601.22154v1

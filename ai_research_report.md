# AI 简报 (2026-06-12)

### Learning to Reason by Analogy via Retrieval-Augmented Reinforcement Fine-Tuning
📄 该论文提出了一种名为检索增强强化微调（RA-RFT）的后训练框架，通过训练检索器按推理获益而非语义相似度排序类比案例，并利用强化学习微调策略模型，使语言模型学会类比推理，从而解决传统RAG在复杂推理任务中检索策略与推理需求不匹配的问题。
🔗 http://arxiv.org/abs/2606.13680v1

### Mana: Dexterous Manipulation of Articulated Tools
📄 本文提出Mana框架，将灵巧操作铰接工具重新定义为动画问题，通过从粗到细的流水线自动生成抓取关键帧并利用运动规划与强化学习转化为操作轨迹，首次实现了从仿真到真实环境下对铰接工具的多指灵巧操作。
🔗 http://arxiv.org/abs/2606.13677v1

### SpatialClaw: Rethinking Action Interface for Agentic Spatial Reasoning
📄 该论文提出**SpatialClaw**，通过重新设计具备灵活性与可组合性的动作接口，使视觉语言模型代理能够根据任务动态调整空间推理策略，突破了现有单一固定执行或结构化工具调用方式的局限性。
🔗 http://arxiv.org/abs/2606.13673v1

# AI 简报 (2026-06-23)

### CoorDex: Coordinating Body and Hand Priors for Continuous Dexterous Humanoid Loco-Manipulation
📄 CoorDex提出了一种协调身体与灵巧手先验的学习框架，将高维控制转化为潜在残差控制，首次实现了人形机器人在连续运动中完成高自由度灵巧操作。
🔗 http://arxiv.org/abs/2606.23680v1

### Semantic Browsing: Controllable Diversity for Image Generation
📄 本文提出一种可控多样性的图像生成方法，支持用户通过有意义的语义变化轴进行结构化浏览与创意探索，解决了传统文本到图像模型在严格遵循提示时多样性不足、且现有增强方法仅产生偶然变体的问题。
🔗 http://arxiv.org/abs/2606.23679v1

### AIR: Adaptive Interleaved Reasoning with Code in MLLMs
📄 该论文提出了一种名为自适应交错推理的方法，通过在增强代码的复杂数值计算任务上扩展强化学习训练，使多模态大模型具备自主选择视觉操作与代码推理的能力，从而突破现有方法仅依赖预定义视觉操作、无法处理数值计算问题的局限。
🔗 http://arxiv.org/abs/2606.23678v1

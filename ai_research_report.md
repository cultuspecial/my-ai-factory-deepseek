# AI 简报 (2026-06-02)

### Mitigating Perceptual Judgment Bias in Multimodal LLM-as-a-Judge via Perceptual Perturbation and Reward Modeling
📄 该论文提出了一种通过**感知扰动与奖励建模**来缓解多模态大语言模型（MLLM）作为评判者时存在的**感知判断偏差**（即倾向奖励看似合理但视觉上错误的答案）的方法，并构建了**感知扰动判断数据集**以支持可验证的评估。
🔗 http://arxiv.org/abs/2606.02578v1

### AdaCodec: A Predictive Visual Code for Video MLLMs
📄 AdaCodec提出了一种面向视频多模态大模型的预测性视觉编码方法，通过仅在场景难以从历史帧预测时发送完整参考帧，否则用紧凑的帧间变化描述（如运动与预测残差）替代冗余的逐帧RGB图像编码，从而显著减少视觉标记的重复与计算开销。
🔗 http://arxiv.org/abs/2606.02569v1

### ClinEnv: An Interactive Multi-Stage Long Horizon EHR Environment for Agents
📄 ClinEnv构建了一个面向长时序住院决策的多阶段交互式医疗环境，通过自动将真实病例转化为有序决策阶段、强制模型主动查询四个专业代理后再提交用药/手术/诊断决策，并利用确定性本体评分模型行为，弥补了现有医疗基准在信息收集、时序决策和可重复评估方面的三重缺陷。
🔗 http://arxiv.org/abs/2606.02568v1

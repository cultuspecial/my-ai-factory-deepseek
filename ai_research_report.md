# AI 简报 (2026-06-29)

### DexCompose: Reusing Dexterous Policies for Multi-Task Manipulation with a Single Hand
📄 DexCompose提出了一种角色感知的残差组合框架，通过显式的指级动作所有权划分解决了多任务灵巧操作中不同技能之间因手指重叠和接触模式冲突导致的干扰问题，实现了用同一只机械手复用预训练策略来执行多个灵巧操作任务。
🔗 http://arxiv.org/abs/2606.28323v1

### Which Nash Equilibrium? Solver-Dependent Selection on Zero-Sum Nash Polytopes
📄 这篇论文的贡献在于：发现解决二人零和博弈的算法并非随机选择纳什均衡，而是系统性地选择均衡集合中的特定成员（如最大熵解），且算法族之间的差异仅在纳什均衡集不对称时显现。
🔗 http://arxiv.org/abs/2606.28308v1

### Agentic Hardware Design as Repository-Level Code Evolution
📄 HORIZON 提出了一种将硬件设计视为仓库级代码演化的自进化智能体框架，首次将EDA软件系统中的仓库级自我演化扩展到硬件设计工件本身，并通过完全免操作的智能体循环在ChipBench、RTLLM、Verilog-Eval等基准测试中实现了100%的完成率。
🔗 http://arxiv.org/abs/2606.28279v1

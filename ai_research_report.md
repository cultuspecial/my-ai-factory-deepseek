# AI 简报 (2026-04-18)

### MM-WebAgent: A Hierarchical Multimodal Web Agent for Webpage Generation
📄 MM-WebAgent提出了一种分层多模态网页生成智能体框架，通过分层规划和迭代自反思协调AIGC元素生成，解决了现有方法因孤立生成元素导致的风格不一致和全局协调性问题，实现了全局布局与局部多模态内容的联合优化，并建立了多模态网页生成基准。
🔗 http://arxiv.org/abs/2604.15309v1

### Generalization in LLM Problem Solving: The Case of the Shortest Path
📄 本文通过构建基于最短路径规划的合成实验环境，发现大语言模型在空间迁移上表现良好，但在路径长度扩展时因递归不稳定性而持续失败，并系统分析了训练数据、训练范式与推理策略对模型系统性泛化能力的影响。
🔗 http://arxiv.org/abs/2604.15306v1

### Diagnosing LLM Judge Reliability: Conformal Prediction Sets and Transitivity Violations
📄 本文提出了一种诊断LLM作为评估者可靠性的双管齐下工具包：通过**传递性分析**揭示了在低总体违规率下仍普遍存在的单输入不一致性，以及通过**共形预测集**为每个实例提供理论保证的可靠性量化指标。
🔗 http://arxiv.org/abs/2604.15302v1
